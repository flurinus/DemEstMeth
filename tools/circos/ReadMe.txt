Circos can be found at www.circo.ca. It is provided here as an extra feature which can be used to produce OD visualization images. It is not developped at EPFL and is an open soure perl project.

Generate circos image by typing in the terminal
./makeimage.sh TestCase_30-Apr-2013_circos.txt

if the name of the file located in the «circos/input/» directory is «TestCase_30-Apr-2013_circos.txt»

Output is generated in circos/img/
